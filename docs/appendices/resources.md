# Additional Resources

This section provides curated list of external resources deepening understanding and exploring advanced topics related to Physical AI and humanoid robotics.

## Books and Publications

*   **"Robotics, Vision and Control" by Peter Corke**: Comprehensive textbook covering robotics fundamentals, kinematics, dynamics, and control, with practical examples in MATLAB and Python.
*   **"Probabilistic Robotics" by Sebastian Thrun, Wolfram Burgard, Dieter Fox**: Foundational text on mobile robot localization, mapping, and navigation using probabilistic methods.
*   **"Deep Learning for Robotics" (Various Authors)**: Collection of research papers and articles exploring deep learning application to robotic perception, control, and reinforcement learning.

## Online Courses and Tutorials

*   **ROS 2 Documentation**: Official documentation for ROS 2 provides extensive tutorials, concepts, and API references.
*   **NVIDIA Isaac Sim Tutorials**: NVIDIA's official tutorials for Isaac Sim cover basic usage, asset creation, and advanced simulation techniques.
*   **Unity Robotics Tutorials**: Official Unity tutorials for integrating robotics simulations with ROS and other tools.
